how to run the Go backendsetup
Step 1 — Copy binary to jump server
scp pdks-security amrv1p000005873:~/
ssh amrv1p000005873
Step 2 — Set environment variables
export SYSDIG_TOKEN=your_token_here
export SYSDIG_BASE=https://us2.app.sysdig.com
export EMAIL_TO=ssachin.kumar@pfizer.com
export SMTP_HOST=smtp.pfizer.com
export SMTP_USER=your_email
export SMTP_PASS=your_password
Step 3 — Run as background service
chmod +x pdks-security
nohup ./pdks-security > pdks.log 2>&1 &
echo $! > pdks.pid
Step 4 — Open dashboard in browser
http://localhost:8080
— or via VPN if accessing remotely
Install as systemd service (survives reboot)
sudo cp pdks-security.service /etc/systemd/system/
sudo systemctl enable pdks-security
sudo systemctl start pdks-security
